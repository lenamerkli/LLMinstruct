§u§
Wie kann ich in der Schweiz gegen LGBTI-feindliche Aussagen vorgehen, die direkt gegen mich gerichtet sind?
§a§
Gegen LGBTI-feindliche Aussagen, die direkt gegen Sie gerichtet sind, können Sie sowohl zivilrechtlich als auch strafrechtlich vorgehen.

\subsubsection{Zivilrecht}
LGBTI-feindliche Äusserungen, die direkt gegen Sie gerichtet sind, gelten als zivilrechtliche Persönlichkeitsverletzung, wenn die Äusserungen Ihr berufliches oder gesellschaftliches Ansehen schmälern. Ihre persönliche Situation und der Kontext der Aussagen müssen berücksichtigt werden: An sich neutrale Bezeichnungen („Schwuler“, „Lesbe“, „Bisexuelle*r“, „Homosexuelle*r“, „Transgender“, „trans Mann“, „trans Frau“, „Intergeschlechtliche*r“) können eine Persönlichkeitsverletzung darstellen, wenn sie in einem homo-, bi-, trans- und/oder interfeindlichen sozialen Umfeld benutzt werden.

Gegen eine Persönlichkeitsverletzung können Sie eine zivilrechtliche Klage einreichen. In einer solchen Klage können Sie beispielsweise verlangen, dass ein Tweet gelöscht wird oder dass offiziell festgestellt wird, dass eine bestimmte Äusserung Ihre Persönlichkeit verletzt hat. Sie können auch fordern, dass dieses Urteil Dritten mitgeteilt oder veröffentlicht wird. Zudem können Sie auf Schadenersatz oder Genugtuung klagen. Eine zivilrechtliche Klage kostet Geld, und beim Einreichen der Klage müssen Sie einen Kostenvorschuss bezahlen. Sie sollten sich deshalb unbedingt juristisch beraten lassen, bevor Sie eine Klage einreichen.

\subsubsection{Strafrecht}
LGBTI-feindliche Äusserungen, die gegen Sie gerichtet sind, können auch strafrechtliche Tatbestände darstellen.

Eine Beschimpfung liegt vor, wenn Sie von einer anderen Person durch schriftliche oder mündliche Aussagen, Gesten oder Handlungen erniedrigend oder verächtlich behandelt werden. Dies ist zum Beispiel der Fall, wenn Sie auf der Strasse angeschrien oder beschimpft werden, weil Sie mit einer Person des gleichen Geschlechts Hand in Hand gehen.

Um üble Nachrede handelt es sich, wenn eine Person Informationen über Sie verbreitet, die Sie vor anderen Personen als verachtenswert erscheinen lassen.

In beiden Fällen handelt es sich um eine Ehrverletzung im strafrechtlichen Sinne. Es muss klar erkennbar sein, dass mit diesen Äusserungen Sie gemeint sind. Die Person, die diese Äusserungen macht oder Handlungen vornimmt, muss die Absicht haben, Sie zu verletzen, oder muss dies mindestens in Kauf nehmen.

Homo- und bi-feindliche Handlungen können ausserdem unter die Strafnorm Art. 261bis StGB (sogenannte Rassismus-Strafnorm) fallen. Dazu müssen folgende Aspekte gegeben sein: Die Äusserung muss einen Aufruf zu Hass oder Diskriminierung wegen der sexuellen Orientierung darstellen und muss öffentlich stattfinden. Dies kann auch online sein.

Gegen solche Äusserungen können Sie bei der Polizei mündlich oder schriftlich einen Strafantrag einreichen. Das Verfahren ist kostenlos.
