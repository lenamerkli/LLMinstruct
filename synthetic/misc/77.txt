To **remove the colored boxes around links** (such as those created by `\cite`, URLs, or section links) in your LaTeX document using the `hyperref` package, you need to modify the `hypersetup` options.

By default, `hyperref` adds colored borders (usually red or blue boxes) around clickable links for visibility in the PDF. These are called **link borders** or **annotations**. To remove them, use the `hidelinks` option.

### âœ… Solution: Add `hidelinks` to `\hypersetup`

Replace or update your current `\hypersetup` block like this:

```latex
\hypersetup{
    breaklinks=true,
    hidelinks
}
```

Or, if you want more explicit control:

```latex
\hypersetup{
    breaklinks=true,
    hidelinks,
    colorlinks=false,     % disables colored links
    pdfborder={0 0 0},    % ensures no border box
}
```

But in practice, `hidelinks` alone is sufficient and commonly used.
