To **remove the colored boxes around links** in your LaTeX document (which are caused by the `hyperref` package), you need to adjust the `hyperref` settings to disable link borders.

By default, `hyperref` places a colored box around links when viewed in some PDF viewers (like Adobe Reader). This is controlled by the `pdfborder` parameter.

---

### âœ… Solution: Add this to your `\hypersetup`

Update your `\hypersetup` block like this:

```latex
\hypersetup{
    breaklinks=true,
    pdfborder={0 0 0},    % Removes the border around links
    linkbordercolor={},   % Optional: explicitly clear link border color
    citebordercolor={},   % Optional: for citations
    urlbordercolor={},    % Optional: for URLs
}
```

### ğŸ” Explanation:
- `pdfborder={0 0 0}` sets the border width of links to zero (no border).
- The other `bordercolor` lines are optional but ensure that no residual styling is applied.

---

### ğŸ“ Additional Tip: If You Want to Keep Borders for Some Links
If you only want to remove borders for **external links** (like URLs), and keep them for internal links (like section references), you can customize more carefully. But for most use cases, `pdfborder={0 0 0}` suffices.
