How can I make that data.legalFirstName is fed by formData.firstName if formData.legalNameDifferent is False?
§a§
You can use a conditional (ternary) operator to set `legalFirstName` based on the value of `legalNameDifferent`. Here's how to modify your code:

```typescript
private handleFormSubmission(formData: AccountFormData): void {
  const data = {
    firstName: formData.firstName,
    lastName: formData.lastName,
    email: formData.email,
    dateOfBirth: formData.dateOfBirth.toISOString().split('T')[0],
    title: formData.title,
    gender: formData.gender,
    country: formData.country,
    legalNameDifferent: formData.legalNameDifferent,
    legalFirstName: formData.legalNameDifferent ? formData.legalFirstName : formData.firstName,
    legalLastName: formData.legalNameDifferent ? formData.legalLastName : formData.lastName,
    legalGender: formData.legalGender,
    identityVerificationInfo: formData.identityVerificationInfo,
  }
  const password = formData.password;
}
```

This way, when `legalNameDifferent` is `false`, `legalFirstName` will be set to `formData.firstName`, and when it's `true`, it will use `formData.legalFirstName`. I also applied the same logic to `legalLastName` since you'll likely want the same behavior for both name fields.
